# 🥊 DefendX

A real-time motion-controlled self-defense game developed using **Python**, **Pygame**, and **OpenCV** as a Final Year Project. DefendX combines computer vision with 2D game development, allowing players to perform combat actions through motion-based controls.

---

## 📖 About the Project

DefendX is an interactive fighting game designed to demonstrate the integration of computer vision and game development. The project uses OpenCV to detect player movements and Pygame to create an engaging combat experience with animated characters, enemy AI, sound effects, and health management.

---

## ✨ Features

- 🎮 Motion-controlled gameplay using OpenCV
- 🥊 Real-time combat mechanics
- 🤖 Enemy AI with attack and dodge behavior
- ❤️ Health bar and round system
- 🔊 Background music and sound effects
- 🎨 Animated sprites and game assets
- ⚡ Smooth gameplay using Pygame

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/alisha-dsz/DefendX.git
cd DefendX
```

### Install dependencies

```bash
pip install pygame opencv-python numpy
```

or

```bash
pip install -r requirements.txt
```

### Run the game

```bash
python main_game.py
```
